
1. 从外部访问 vue 项目, 在 webpack.config.js 中设置:
```
 devServer: {
    // 在第1行设置
    host: '0.0.0.0', // https://webpack.docschina.org/configuration/dev-server/#devserverhost
    ...
}
```

2. 在浏览器输入地址, 就可以在局域网内进行访问, 浏览器地址为:
```
http://192.168.1.2:8080/#/register
```

3. 电信宽带 TY-6201A 的外网访问:
```
1). 查询 ip 公网地址: 
    可以通过 以下方法查询到:
    http://192.168.1.1 进入天邑 TY-6201A 路由器, 在路由状态 下点击 互联网, 可以看到 ipv4 信息下的 ip地址就是
    公网的ip地址
    我在天邑路由器上做了如下地址绑定, 以便本地ip固定在 192.168.1.2这个地址   
    DHCP静态IP分配: 
       IP地址      	      MAC地址	                 
    192.168.1.2	     d0:37:45:92:65:14	 
2). 高级设置---> 应用设置---> 端口转发---> 添加
    名称           协议:    外部端口     内部IP地址       内部端口 
    lynn-pc1      TCP        8080      192.168.1.2        8080
    lynn-pc1      TCP        3000      192.168.1.2        3000
3). 这样设置不成功, 广域网无法访问本机 web 服务器, 问题出在什么地方呢?
```
